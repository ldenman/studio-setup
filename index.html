Lake's Studio

Neve preamp, 1176 compressor, LA-2A leveling amp — normalled through a 48-point patchbay. Wing preamp for gain, outboard for color.

Analog · Digital · Automated

The Studio

The studio should never get in the way of the music. Every input runs through a permanently wired analog chain — normalled, gain-staged, ready. Plug in and hit record. The outboard is already in the path.

The Wing Rack handles routing, monitoring, and 16 slots of classic effect emulations, all controllable via OSC. Logic Pro session players route through the console alongside live instruments. Everything records dry to the Tascam Model 12 — clean multitracks, analog monitoring.

What happens here

Recording

Live instruments through Neve preamp, 1176, and LA-2A. Dry capture to multitrack. Analog monitoring.

Amp sims, tape saturation, pitch shifting, modulation. AmpliTube, TONEX, and MixBox in the DAW.

Outboard

Four units, permanently wired. Neve preamp for color and EQ shaping. 1176 for fast transient control. Opto for smooth vocal leveling. Distressor in the default guitar chain after the 1176. Patch cables to re-route.

HA73-EQX2 — Neve 1073 Preamp/EQ Clone (Heritage Audio)

Dual-channel transistor preamp with 3-band EQ. Carnhill transformers. In the default guitar and vocal chains.

EQ: Low shelf (35/60/110/220 Hz), Mid peak (360/700/1.6k/3.2k/4.8k/7.2k Hz), High shelf (10k/12k/16k Hz), HPF (50/80/160/300 Hz)

I/O per channel: XLR mic in, XLR line in, 1/4" Hi-Z DI (front), XLR output

WA76-D2 — 1176 FET Compressor Clone (Warm Audio)

Dual-channel FET compressor. CineMag transformers. Dry/wet knob for parallel compression. Three stereo modes: Linked, Primary/Secondary, Dual Mono.

Controls: Input (threshold), Output (makeup), Attack (20μs–800μs), Release (50ms–1.1s), Ratio (4:1, 8:1, 12:1, 20:1, ALL buttons)

I/O per channel: XLR + 1/4" TRS input, XLR + 1/4" TRS output

Distressor — Compressor (Empirical Labs EL8)

Single-channel compressor. In the default guitar chain after the WA76 B (P7–P8). Transformerless balanced I/O.

Controls: Input, Output, Attack, Release, Ratio (1:1 to NUKE), Detector HP, Dist 2 (tape), Dist 3 (tube)

I/O: XLR + 1/4" TRS input, XLR + 1/4" TRS output, 1/4" TRS stereo link

Audioscape Opto — LA-2A Optical Compressor Clone

Single-channel optical compressor. In the default vocal chain after the 1176. Smooth, musical, set-and-forget.

Controls: Peak Reduction, Gain, Mode (Compress/Limit)

I/O: XLR input, XLR output, 1/4" TRS stereo link

Recorder

Tascam Model 12. 12 tracks to SD card via Loopback — no physical audio cables. Dry capture while you monitor through the analog chain.

Tascam Model 12 — Multitrack Recorder / Interface

12-track recorder, 12-in/10-out USB-C audio interface. Connected via Loopback software — no physical audio cables. Records to SD card.

Physical I/O (unused): 10x XLR/TRS combo inputs (all with phantom), XLR main outs, sub outs, 2 AUX sends, 2 headphone outs, click out, footswitch, MIDI in/out, Bluetooth

Patchbay

Samson S-Patch Plus. 48 points, half-normalled. Default chains are always connected — insert a cable to re-route.

Signal Flow

Vocal mic hits the Wing preamp for gain, then sends out to the outboard chain via a bus. No patching required — it's all normalled. Swap gear by inserting a cable.

Sounds

Starting points. Every session gets dialed to taste.

Vocal Recording (Outboard)

Wing preamp for gain, then out through the analog chain for color and control.
