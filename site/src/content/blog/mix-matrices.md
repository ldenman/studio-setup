The studio had a routing problem. We needed to send processed stems from the Wing to the Model 12 for hands-on mixing, but we'd run out of buses. Most were already spoken for — outboard sends, recording paths, monitoring, reverb.

## The Solution Nobody Talks About

The Wing has eight matrix outputs. Matrices are designed for exactly this — creating independent mixes for different destinations. One was already driving the speakers. The other seven were empty.

So we wired each matrix to a Model 12 channel: vocal, rhythm guitar, lead guitar, bass, drums, piano, and a spare for overdubs. The matrices became the permanent bridge between the two consoles. Change the source on the Wing per project; the Model 12 side never moves.

## Why This Changes Everything

Fader 1 on the Model 12 is always vocal. Fader 2 is always rhythm guitar. You sit down, and the console is already laid out the same way every time. No reconfiguring, no guessing which channel is where.

The Wing handles which track feeds which matrix and adds any processing upstream — amp sims, doublers, octavers. The Model 12 just mixes: real faders, analog EQ, compression. Every pass is captured automatically on a stereo track. Move a fader, run it again. Mixing becomes a performance.

## The Bigger Idea

This is the studio's design philosophy in miniature: wire it once, and every session starts the same way. The creative energy goes into the music, not the routing.
