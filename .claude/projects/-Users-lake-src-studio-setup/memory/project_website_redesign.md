The studio website (index.html) is being redesigned from a pure technical reference into a proper studio website. WEBSITE.md contains the full plan.

**Why:** The current site reads like documentation — channel numbers and OSC commands. It needs to feel like a studio where music gets made, while preserving the excellent technical reference content underneath.

**How to apply:** When working on the website, follow the goals and section order in WEBSITE.md. Keep it as a single vanilla HTML/CSS/JS file. Dark theme stays. Technical sections become a deeper layer, not the front door. Hero → identity → offerings → gear with context → signal flow → sounds → effects catalog → tools → contact.
